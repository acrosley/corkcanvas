# Cork & Canvas Co. Website

A premium, responsive website for Cork & Canvas Co. showcasing custom hand-painted bottles and banners.

## Features

- **Premium Design**: Modern aesthetics with glassmorphism and smooth animations.
- **Dynamic Gallery**: Filterable portfolio of custom work.
- **Pricing Guide**: Clear breakdown of services and add-ons.
- **Order Inquiry**: Integrated contact form for customers.

## Deployment to Render

This project is configured for easy deployment to [Render](https://render.com/).

### Option 1: Automatic (Recommended)

1. Push this code to a GitHub repository.
2. Log in to Render and click **New +** > **Blueprint**.
3. Connect your GitHub repository.
4. Render will automatically detect the `render.yaml` file and set up your static site.

### Option 2: Manual

1. Log in to Render and click **New +** > **Static Site**.
2. Connect your GitHub repository.
3. Set the **Build Command** to be empty.
4. Set the **Publish Directory** to `.`.
5. Click **Create Static Site**.

## Local Development

Simply open `index.html` in your browser or use a local server like `Live Server` in VS Code.
